## Install
1. Download `the-dictator-{{VERSION}}-arm64.dmg`.
2. Verify checksum:
   - `shasum -a 256 the-dictator-{{VERSION}}-arm64.dmg`
   - Compare with `the-dictator-{{VERSION}}-arm64.sha256`.
3. Open DMG and drag `the-dictator.app` to Applications.
4. Launch from `/Applications`.
